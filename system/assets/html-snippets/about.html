Behind DUFT

DUFT is developed by the University of California, San Francisco. It is built on a clean, modular architecture that separates the backend data engine from the visual frontend and configuration layer. The backend is written in Python, making it accessible to data teams already familiar with scripting and Jupyter Notebooks.

Data transformation pipelines are executed as scheduled ETL jobs or on demand, connecting to a wide range of Python-compatible databases including PostgreSQL, MySQL, SQL Server and SQLite. The frontend is built using ReactJS, delivering a fast and responsive user experience for dashboards. Visualisations use industry-grade libraries like ApexCharts, with support for filters, search, data exports, and layout customisation.

Core Architecture

Built with Python and ReactJS, featuring a modular design that separates backend data processing from frontend visualization. Database agnostic and AI-ready for natural language queries.

Customization Layer

Easy configuration through JSON files - no coding required. Define dashboards, indicators, and filters with reusable components that can be updated without touching application code.

Deployment & Operations

Flexible deployment options with Git integration and enterprise-grade security features. Run as a central server or standalone application with minimal resource requirements.

What DUFT Can Do

Make EMR Data Easy to Report On

Extract, transform, and present data from systems like OpenMRS or ERPNext clearly and longitudinally—without needing to modify the EMR itself.

Automate NDR Reports

Automate submissions to National Data Repositories by mapping your local data into the expected format and uploading it reliably on schedule.

Remote Facility Updates

Enable central teams to push updates—add charts, fix bugs, or update transformations—without physical site visits, saving time and resources.

AI-Powered Insights

Surface better insights using lightweight AI to assist with anomaly detection, data summarization, and trend identification for improved decision-making.

Multi-Source Data Integration

Combine data from multiple sources, whether at site or national level, to visualize and analyze information holistically through custom integration scripts.

User-Friendly Dashboards

Build attractive, interactive dashboards using an integrated HTML-like language without needing to learn complex tools, with remote update capabilities.
